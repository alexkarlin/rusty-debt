## What is rusty-debt?

Rusty-debt is an educational project created as an introduction to the Rust language. It allows you to track in- and outgoing debt, so youâ€™ll never miss out on another heated argument with your friend Joe about the 20 bucks he borrowed but never paid you back.

## How to build rusty-debt

1. Clone this repository
2. Open a terminal window in the root directory of rusty-debt
3. Run `cargo build`

The output directory (_/debug_ or _/target_ by default) will now have a rusty-debt executable in it).
